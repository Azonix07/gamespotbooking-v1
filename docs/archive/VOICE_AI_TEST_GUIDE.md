## üêõ Troubleshooting

### **Issue: Microphone not starting**
- **Solution**: Check browser permissions
- Chrome: Settings ‚Üí Privacy ‚Üí Microphone
- Allow microphone access for localhost

### **Issue: Background not blurred**
- **Check browser**: Chrome/Safari work best
- Firefox has limited blur support
- Try refreshing page (Cmd+Shift+R)

### **Issue: Sphere not vibrating much**
- **Solution**: Speak louder
- Check microphone volume in system settings
- Try getting closer to microphone

### **Issue: Voice sounds robotic**
- **Check console**: See which voice is being used
- Chrome: Should use "Google" voices
- Safari: Should use "Samantha" or "Karen"
- May need to download system voices

### **Issue: Not auto-starting**
- **Wait 1 second**: 800ms delay is intentional
- Check status indicator for "Listening..."
- Try refreshing page
